# 🛡️ SecRoad: CyberSecurity Learning Path 💀

**SecRoad** é uma plataforma gamificada de aprendizado em Cibersegurança e Infraestrutura de Redes. Desenvolvido para transformar o estudo teórico em uma experiência de "Incidence Response", onde cada lição concluída aproxima o usuário da elite do comando digital.

---

## 🚀 Funcionalidades Principais

* **Deca-Tier Ranking:** Sistema de progressão com 9 patentes exclusivas, de *Support Level 1* até *Ghost Commander*.
* **Emergency Protocol (System Breach):** Eventos aleatórios de alta prioridade (60% spawn rate) que desafiam o conhecimento do usuário em situações críticas.
* **Economia de BITS:** Ganhe recompensas por completar desafios e suba no ranking global.
* **Interface Terminal-Aesthetic:** Design inspirado em ambientes de alto desempenho (Zorin OS/Arch Linux).
